Nashville Housing

Leveraging raw Nashville housing market data, I conducted comprehensive data cleaning in SQL Server to enhance its usability for analysis. This involved standardizing date formats, filling in missing data, splitting columns, removing duplicates, rectifying data inconsistencies, and eliminating unused columns. Subsequently, I performed Exploratory Data Analysis (EDA) to extract meaningful insights into the number of properties per city, properties per owner, sold properties per year, and price categories.
